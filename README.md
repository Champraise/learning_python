# 🐍 Introducing Python: Revision Repo

This repository tracks my structured revision of Python fundamentals using the book **"Introducing Python"** by Bill Lubanovic.

My goal is to sharpen my core Python skills—specifically focusing on clean code, data structures, and systems programming—to support my transition into Data and AI Engineering.

## 📂 Contents

The code here is organized by the book's chapters, covering:

- **Basics & Data Structures** (Variables, Lists, Dictionaries, Sets)
- **Control Flow & Functions** (Loops, Comprehensions, Generators)
- **Modules & OOP** (Classes, Objects, Packages)
- **Data Wrangling** (File I/O, SQL Databases, APIs)
- **Systems & Concurrency** (Multiprocessing, Async)
