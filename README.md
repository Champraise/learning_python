# Learning Python

This repository contains my journey through learning Python using the "Learning Python" book (or any other resources I might find fun). It's a place to store code examples, exercises, and personal projects as I progress.

The repository is organized into directories based on chapters or topics from the learning materials:

*   `chapter1/`: Contains code examples and exercises from Chapter 1.
*   `chapter2/`: Contains code examples and exercises from Chapter 2.
*   `exercises/`: Contains solutions to exercises.
*   `projects/`: Will contain small projects I create to practice my skills.
